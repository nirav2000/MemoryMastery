# Reusable Memory Mastery specialist agents

These role prompts are stored so Codex can reuse the same quality-review agents in this repository and copy them into other learning apps.

Use them when a task asks for product review, visual QA, accessibility review, content review, information architecture, or front-end architecture. The lead agent should still own final decisions, implementation, testing, commit and push.

## Available roles

1. `ux-researcher.md` — first-time learner comprehension and abandonment risks.
2. `information-architect.md` — route map, feature progression and navigation clarity.
3. `visual-quality-agent.md` — layout, spacing, contrast, screenshots and small detail defects.
4. `accessibility-reviewer.md` — keyboard, semantics, labels, focus, contrast and motion.
5. `content-designer.md` — learner-facing language, tone and terminology.
6. `front-end-architect.md` — rendering structure, CSS conflicts, state, storage and testability.

Each role must provide concrete route/selector references and must not declare success without rendered inspection when visual output is involved.
